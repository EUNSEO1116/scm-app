# 시스템 및 데이터

## 개요
Google Sheets 연동, 엑셀 처리, 데이터 저장 등 앱 전반의 기반 시스템.

---

## 1. 시트 동기화 (sheetSync.js)
**경로:** `src/sheetSync.js`

### Google Sheets 연동
- 스프레드시트 ID: `1NXhW_gG0b-gXuVqrhbY9ErWi8uO_7pXIy-NTo4FbE1I`
- CSV/TSV 엔드포인트로 시트 데이터 읽기
- Apps Script 엔드포인트로 시트 데이터 쓰기

### 품절 기록 동기화
| 함수 | 기능 |
|---|---|
| `fetchFromSheet()` | '품절 기록' 시트 CSV 읽기 |
| `saveReasonsToSheet(items)` | 품절 사유 시트에 쓰기 |
| `deleteReasonFromSheet(barcode)` | 품절 사유 삭제 |

### 주의 품목 동기화
| 함수 | 기능 |
|---|---|
| `fetchCautionItems()` | '주의 품목' 시트 읽기 |
| `saveCautionItem(barcode, productName, optionName)` | 주의 품목 등록 |
| `deleteCautionItem(barcode)` | 주의 품목 삭제 |

### 보호 로직
- `markLocalSave(barcodes)` — 로컬 저장 시각 기록
- `getProtectedBarcodes()` — 10분 이내 저장된 바코드 목록
- `PROTECT_MINUTES = 10` — 시트 덮어쓰기 방지 시간

### Apps Script 엔드포인트
```
POST https://script.google.com/macros/s/.../exec
Body: { action, barcode, items, ... }
```

## 4. localStorage 데이터 맵

앱 전반에서 사용하는 localStorage 키와 연관 기능:

| 키 | 용도 | 사용처 |
|---|---|---|
| `fbc_savings_history` | FBC 비용 계산 이력 | FBC 계산기 → 대시보드, 홈 |
| `soldout_rate_snapshots` | 일별 품절률 스냅샷 | 품절률 |
| `soldout_exclude_items` | 품절 제외 품목 | 제외관리 → 품절현황, 품절률, 재고계산기 |
| `soldout_reasons_v2` | 품절 사유 | 품절현황 ↔ 품절기록 |
| `soldout_history` | 품절 이력 | 품절현황 ↔ 품절기록 |
| `soldout_prev_barcodes` | 전일 품절 바코드 | 품절현황 |
| `new_product_stock` | 신규상품 재고 | 품절현황 |
| `local_special_items` | 로컬 특별관리 항목 | 이슈관리 → 발주장부, 홈 |
| `orderbook_notes` | 발주장부 메모 | 발주장부, 홈 |
| `supplies_orders` | 부자재 발주 데이터 | 부자재발주 → 부자재목록 |

## 5. 데이터 흐름도

```
GOOGLE SHEETS (원본 데이터)
├── 재고 계산기 ──→ 재고계산기, 발주신청, 품절현황, 품절률, 입고신청
├── 데이터 입력 ──→ 재고계산기, 품절현황, 품절률, 매출관리
├── 쿠팡바코드 ──→ 발주신청, 입고신청
├── 발주장부 ──→ 발주장부, 품절현황, 부자재목록
├── 발주서 양식 ──→ 발주신청
├── 품목 FBC ──→ FBC 품목
├── 일일 판매량 ──→ 매출관리
├── 특별 관리 상품 ──→ 이슈관리
├── 부자재목록 최신 ──→ 부자재목록
├── 품절 기록 ──→ 품절기록 (sheetSync)
└── 주의 품목 ──→ 매출관리 (sheetSync)

localStorage (로컬 영속)
├── fbc_savings_history: FBC계산기 → 대시보드, 홈
├── soldout_*: 품절현황 ↔ 품절기록 ↔ 품절률 ↔ 제외관리
├── local_special_items: 이슈관리 → 발주장부, 홈
├── orderbook_notes: 발주장부 ↔ 홈
└── supplies_orders: 부자재발주 → 부자재목록

Apps Script (쓰기 게이트웨이)
├── 품절 사유 저장/삭제
└── 주의 품목 등록/삭제
```
