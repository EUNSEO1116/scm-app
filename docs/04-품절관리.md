# 품절관리

## 개요
품절 현황 모니터링, 품절률 추적, 제외 품목 관리, 품절 이력 기록 기능군.

---

## 1. 품절 현황 (SoldOut.jsx)
**경로:** `/soldout`

### 기능
- 현재 품절/위험 품목 실시간 모니터링
- 품절 사유 입력 및 시트 동기화
- 엑셀 내보내기

### 데이터 소스
- Google Sheets: 재고 계산기, 데이터 입력, 발주장부
- localStorage: `soldout_reasons_v2`, `soldout_history`, `soldout_prev_barcodes`
- sheetSync.js: 품절 기록 시트 읽기/쓰기

### 상태 분류
| 상태 | 표시 |
|---|---|
| 🔴 품절 | 재고 0 |
| 기타 위험 등급 | 재고 기준별 분류 |

### 주요 기능
- **품절 사유 입력** — 선택한 품목에 사유 작성, 시트 동기화
- **품절률 표시** — 상단에 현재 품절률 (유효품목 대비)
- **이전 대비 변동** — 전일 품절 바코드와 비교, 신규/해소 표시
- **다중 선택** — 체크박스로 일괄 사유 입력
- **신규상품 재고 추적** — `new_product_stock` localStorage

### 제외 로직
- '최종마감', '품질확인서', '마감대상' 키워드 자동 제외
- `soldout_exclude_items` (품절 제외 관리) 참조

### 연관 기능
- **품절률** — 품절 수 데이터 공유
- **품절 기록** — 사유/이력 데이터 공유 (`soldout_history`, `soldout_reasons_v2`)
- **품절 제외** — 제외 목록 참조
- **재고 계산기** (`01-재고관리.md`) — 동일 제외 목록 사용

## 2. 월별 품절률 (SoldOutRate.jsx)
**경로:** `/soldout/rate`

### 기능
- 매일 페이지 방문 시 자동 스냅샷 저장
- 월별 품절률 추이 차트
- 일별 기록 테이블

### 자동 스냅샷
- 오늘 품절 수 / 전체 품목 수 → 품절률 계산
- localStorage `soldout_rate_snapshots`에 저장

### 통계 카드
| 항목 | 설명 |
|---|---|
| 오늘 품절률 | 당일 스냅샷 |
| 이번달 평균 품절률 | 월 내 일별 평균 |
| 전체 평균 품절률 | 전체 기간 평균 |
| 목표 품절률 | 5% |

### 월별 차트
- 연도 선택 네비게이션
- 색상: ≤5% 초록, 5~10% 주황, >10% 빨강
- 호버 시 평균 품목/품절률 표시

### 일별 기록
- 컬럼: 날짜, 전체 품목, 품절 수, 품절률
- 날짜 내림차순 정렬
- 전체 삭제 가능 (확인 대화상자)

### 제외 로직
- `soldout_exclude_items` 참조하여 제외 품목 품절률에서 빼기
- 신규상품 중 입고 예정 있으면 제외

## 3. 제외 품목 관리 (SoldOutExclude.jsx)
**경로:** `/soldout/exclude`

### 기능
- 품절률 계산에서 제외할 품목 관리
- 종료일 기반 자동 만료
- 만료 항목 별도 관리

### 제외 항목 등록
- 상품 검색 (이름/바코드)
- 다중 선택 → 종료일 + 사유 입력 → 일괄 등록
- 종료일 미설정 시 무기한 제외

### 활성 제외 목록
- 남은 일수 표시 (3일 이하: 빨간색)
- 검색 결과 최대 100건

### 만료 제외 목록
- 종료일 경과 항목 별도 표시
- 만료 기록 일괄 삭제

### 데이터 저장
- localStorage `soldout_exclude_items`

### 연관 기능
- **품절 현황** — 제외 목록 참조
- **품절률** — 제외 목록으로 품절률 보정
- **재고 계산기** (`01-재고관리.md`) — 동일 제외 목록 사용

## 4. 품절 기록 (SoldOutHistory.jsx)
**경로:** `/soldout/history`

### 기능
- 품절 이력 추적 (상품별 그룹)
- Google Sheets '품절 기록' 시트 동기화
- 제외 목록으로 이동 기능

### 데이터 동기화
- `fetchFromSheet()` — 시트 CSV 읽기
- localStorage `soldout_history`, `soldout_reasons_v2`와 병합
- 최근 저장(10분 이내) 보호 — 시트 데이터로 덮어쓰기 방지

### 상품별 이력
- 바코드별 그룹화
- 상품명, 옵션명, 바코드 표시
- 품절 날짜 인라인 편집

### 제외 이동
- 품절 항목 → 제외 목록으로 이동 버튼
- 종료일 선택 필수, 중복 방지

### 통계
- 이번달 품절 품목 수
- 반기 누적 품목 수

### 엑셀 내보내기
- 컬럼: 바코드, 상품명, 옵션명, 품절일, 사유

### 연관 기능
- **품절 현황** — `soldout_history`, `soldout_reasons_v2` 공유
- **제외 관리** — `soldout_exclude_items` 쓰기
