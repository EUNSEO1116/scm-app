# 재고관리

## 개요
재고 현황 분석, 입고 배분, 발주 신청/관리를 담당하는 핵심 기능군.

---

## 1. 재고 계산기 (InventoryCalculator.jsx)
**경로:** `/inventory`

### 기능
- Google Sheets(재고 계산기 + 데이터 입력 시트)에서 실시간 데이터 로드
- 알림 시스템으로 재고 상태 자동 분류
- 엑셀 내보내기

### 주요 컬럼
| 컬럼 | 설명 |
|---|---|
| 알림 | 긴급/잠재긴급/과잉주의/과잉재고/정상/판매없음 |
| 쿠팡재고 | 쿠팡 센터 보유 재고 |
| 입고 | 그로스입고예정 + 입고 |
| 박스히어로 | BoxHero 창고 재고 |
| 총재고 | 전체 합산 재고 |
| 3일평균판매 | 최근 3일 평균 판매량 |
| 발주량 | 3/7/30일 기준 발주 추천량 |
| 예상판매주 | 총재고 / 일평균판매 / 7 |
| 안전재고 | 일평균판매 × 리드타임 |

### 알림 기준
- 🚨 **긴급** — 총재고 < 14일분 판매량 (3일평균판매가 0인 품절 상품은 제외 → '판매없음'으로 분류)
- 🚨 **잠재긴급** — 긴급에 근접
- 🔶 **과잉주의** — 재고 과다 경고
- 📦 **과잉재고** — 재고 > 30일 판매량 × 3
- ✅ **정상** — 예상판매 6~9주
- — **판매없음** — 판매 데이터 없음

### 제외 로직
- 키워드 자동 제외: '최종마감', '품질확인서', '마감대상'
- 사용자 정의 제외 (localStorage `soldout_exclude_items` 참조)

### 연관 기능
- **품절 제외 관리** (`04-품절관리.md`) — 제외 품목 목록 공유

## 2. 입고신청 (Incoming.jsx)
**경로:** `/inventory/incoming`

### 기능
- FBC 출고내역서(엑셀) 업로드
- 입고 물량을 쿠팡 vs 박스히어로로 자동 배분
- 배분 결과 엑셀 내보내기 (색상 구분)
- 주문목록 업로드 (윙 발송 체크)
- 대기필요 바코드 등록
- VOC 자동 표시 (상품개선 DB 연동)
- 특별관리 자동 표시 (특별관리 시트 연동)

### 배분 로직
- 목표재고 = 6주 × 주간판매량
- 목표치까지 쿠팡 배분, 나머지 박스히어로
- "NEW" 발주 → 강제 박스히어로 배분

### 주간판매량 계산 (입고신청 260406버전)
- 재고계산기 시트의 "쿠팡재고+입고예정 예상판매주"(col[21])로 주간판매량을 역산
- `stockAll = 재고(G열/col[6]) + 그로스입고예정(H열/col[7]) + 입고(I열/col[8])`
- `weeklySales = stockAll / 예상판매주` (예상판매주 > 0일 때)
- `needed = 6주 목표재고 - stockAll` 만큼 쿠팡 배분, 나머지 박스히어로
- **판매량 없음(weeklySales ≤ 0)** → 무조건 박스히어로 배정
- 엑셀 출고수량(E열), 합계:출고수량(K열)은 숫자 형식 유지 (드래그 시 합계 표시)

### 색상 코드
| 색상 | 의미 |
|---|---|
| 노랑 (#FFFF00) | 고양/시흥 (쿠팡 센터) |
| 초록 (#C6EFCE) | 경기광주/안성 |
| 회색 (#F2F2F2) | 박스히어로 |

### 주문목록 업로드 (윙 발송 체크)
- `전계정_주문목록_날짜.xlsx` 파일 업로드
- 엑셀 열 매핑: D열(주문수량), E열(상품명), F열(옵션명)
- E열(상품명) + F열(옵션명)을 재고계산기 시트에서 찾아 SKU로 변환
- 같은 SKU의 주문수량을 합산하여 비고에 `N개 윙 발송` 표시 (파란색 배지)

### 대기필요 등록
- 바코드(SKU)를 복사붙여넣기로 입력 (줄바꿈, 쉼표, 탭 구분)
- 등록된 바코드가 입고 목록에 있으면 비고에 `대기필요` 표시 (주황색 배지)
- 개별 삭제 / 전체 삭제 가능

### VOC 자동 표시
- 상품개선 DB(`improvement_items`)에서 유형이 `CSV·VOC`이고 상태가 `시작전` 또는 `처리중`인 항목 조회
- 상품명 키워드 매칭: 입고 목록의 라벨명에 상품개선 상품명이 포함되면 비고에 `VOC` 표시 (빨간색 배지)

### 특별관리 자동 표시
- 특별관리 시트의 기타(1회성)에 값이 있는 바코드 → 비고에 `특별관리` 표시 (보라색 배지)

### 비고 컬럼 (G열)
| 조건 | 표시 예시 | 배지 색상 |
|------|-----------|-----------|
| 주문목록 매칭 | `5개 윙 발송` | 파란색 (#e8f0fe) |
| VOC (상품개선 DB) | `VOC` | 빨간색 (#fce4ec) |
| 특별관리 (기타1회성) | `특별관리` | 보라색 (#f3e5f5) |
| 대기필요 등록 | `대기필요` | 주황색 (#fff3e0) |

엑셀 다운로드 시에는 쉼표로 합쳐서 텍스트로 출력 (예: `5개 윙 발송, VOC, 특별관리, 대기필요`)

### 데이터 소스
- 사용자 업로드 엑셀 + 쿠팡바코드 시트 + 재고계산기 시트 교차 참조
- 재고계산기 시트에서 상품명+옵션명 → SKU 매핑 테이블 생성 (주문목록 매칭용)
- 특별관리 시트: 기타(1회성) 존재 여부 확인 (특별관리 배지용)
- 상품개선 DB(`improvement_items`): CSV·VOC 유형 + 시작전/처리중 항목 조회 (VOC 배지용)

## 3. 발주신청 (OrderRequest.jsx)
**경로:** `/inventory/order`

### 기능
- 재고계산기에서 CN발주량(Q열) > 0인 항목 자동 추출
- 브랜드 코드 기반 발주번호 자동 생성
- 클립보드 복사 (TSV) / 엑셀 내보내기

### 발주번호 규칙
- 형식: `{브랜드코드}-{YYMMDD}{-JJ}`
  - 예: `AE-I-260401-JJ` (긴급 발주 시 -JJ 접미사)
- 경기광주/안성 센터는 코드 뒤에 '2' 추가
- 주간판매 < 3.5주분이면 `-JJ`(긴급) 표시

### 브랜드 코드 매핑
- 일상포인트 → AE-I, 하루모음 → AE-HM 등

### 주요 필드
- SKU, 상품명, 발주량, 입고센터
- 봉제여부, FBC품목, 기타(1회성), 발주단위

### 연관 기능
- **이슈관리** (`05-이슈관리.md`) — 특별관리 상품 정보 참조 (봉제, FBC, 발주단위 등)
- **부자재관리** (`06-부자재관리.md`) — 발주 시 부자재 소모량 연동
