# 이슈관리

## 개요
특별 관리가 필요한 상품(봉제, FBC, 가격 협의 등)을 등록하고 추적하는 기능.

---

## 1. 이슈관리 (IssueManagement.jsx)
**경로:** `/issue`

### 데이터 소스
- Google Sheets '특별 관리 상품' 시트 (Apps Script 경유)
- localStorage `local_special_items` (로컬 임시 등록)

### 주요 컬럼
| 컬럼 | 설명 |
|---|---|
| 바코드 | 쿠팡 바코드 |
| 등록상품명 | 상품명 |
| 옵션명 | — |
| 비고 | 특이사항 메모 |
| 발주단위 | 발주 시 단위 |
| 봉제여부 | 봉제 필요 여부 |
| FBC품목 | FBC 카테고리 |
| 가격 협의 | 가격 관련 메모 |
| 기타(1회성) | 일회성 특이사항 |
| 메모용 | 자유 메모 |

### 검색 및 필터
- 상품명/바코드/비고 텍스트 검색
- 브랜드별 필터
- 컬럼 정렬

### 로컬 항목 등록
- 시트 반영 전 로컬에서 먼저 등록 (localStorage)
- 시트에 동일 바코드 존재 시 동기화 알림 표시
- "적용 완료" 버튼 → localStorage에서 제거

### 동기화 알림
- localStorage에 있는데 시트에도 바코드가 있는 항목 감지
- 주황색 경고 박스로 사용자에게 안내
- 시트에 수동으로 메모 추가 유도

### 셀 확장
- 긴 텍스트 셀 클릭 → 펼쳐서 전체 내용 표시

### 연관 기능
- **발주신청** (`01-재고관리.md`) — 발주 시 봉제여부, FBC품목, 발주단위 등 참조
- **발주장부** (`01-재고관리.md`) — 특별관리 항목 표시에 `local_special_items` 사용
- **홈** — `local_special_items` 데이터 읽어감
