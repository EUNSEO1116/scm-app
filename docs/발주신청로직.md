# 발주신청 (OrderRequest.jsx)

## 개요
재고계산기 시트에서 CN발주량(Q열) > 0인 상품을 자동 추출하여 발주서를 생성하는 페이지.

- 경로: `/inventory/order`
- 컴포넌트: `src/pages/OrderRequest.jsx`
- 메뉴 위치: 재고관리 > 발주신청

---

## 데이터 소스

### 1. 재고계산기 시트 (TSV)
- Q열(col[16]) = CN발주량 → 수량 > 0인 행만 추출
- 가져오는 값: SKU(col[2]), 노출상품명(col[3]), 옵션명(col[4]), 총재고 예상판매주(col[22])

### 2. 쿠팡바코드 시트 (CSV)
- 바코드(col5) → 브랜드(col8), 입고센터(col11), 비고(col13) 매핑

### 3. 발주서 양식 시트 (CSV)
- SKU(col2) → 비고(col4) 매핑
- 비고 우선순위: 발주서 양식 비고 > 쿠팡바코드 비고

### 4. 특별 관리 상품 시트 (CSV)
- SKU(col0) → 봉제여부(col5), FBC품목(col6), 기타1회성(col8) 매핑

---

## 발주번호 생성 규칙

### 형식
`{브랜드코드}-{YYMMDD}{-JJ}`

### 브랜드 코드 매핑
| 브랜드 | 코드 |
|--------|------|
| 일상포인트 | AE-I |
| 하루모음 | AE-HM |
| 리빙스타일 | AE-L |
| 어리플 | AE-S |
| 룸앤업 | AE-R |
| 펄빈 | AE-P |
| 생활기준 | AE-SH |
| 토글리 | AE-T |
| 에브리잇템 | AE-E |
| 프루드 | AE-E |
| 로즈바운드 | AE-B |
| 원데이홈 | AE-O |
| (매핑 없음) | AE-X |

### 센터 보정
- 경기광주 또는 안성 센터 → 코드 뒤에 `2` 추가 (예: `AE-I2`)

### 긴급 발주
- 총재고 예상판매주(col[22]) < 3.5주 → `-JJ` 접미사 (예: `AE-I-260506-JJ`)

## 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| 발주번호* | 자동 생성된 발주번호 |
| 한글옵션명* | 노출상품명 + 옵션명 |
| SKU* | 쿠팡바코드 |
| 수량* | CN발주량 |
| 비고 | 발주서 양식 또는 쿠팡바코드의 비고 |
| 실패 원인 | (빈칸, 수동 입력용) |
| 입고센터 | 쿠팡바코드 시트의 입고센터 |
| 봉제 | 특별관리 시트의 봉제여부 (있으면 빨간색) |
| FBC | 특별관리 시트의 FBC품목 (있으면 파란색) |
| 기타(1회성) | 특별관리 시트의 기타1회성 (있으면 주황색) |

## 기능

### 1. 발주번호별 그룹 카드
- 같은 발주번호끼리 묶어서 카드로 표시
- 카드 헤더에 브랜드, 센터, 품목수, 총수량 표시

### 2. 복사 (TSV 클립보드)
- **그룹별 복사**: 카드 단위로 탭 구분 복사
- **전체 복사**: 모든 발주 항목 한번에 복사
- 복사 컬럼: 발주번호, 상품명, SKU, 수량, 비고, (빈칸), 센터, 봉제, FBC, 기타1회성
- 비고 내 줄바꿈은 공백으로 치환

### 3. 엑셀 다운로드
- 파일명: `CN발주서_YYMMDD.xlsx`
- 시트명: `주문서엑셀양식`
- 컬럼: 발주번호*, 한글옵션명*, SKU*, 수량*, 비고, 실패 원인, 입고센터
- xlsx-js-style로 스타일 적용 (헤더 볼드, 가운데 정렬)

### 4. 복사 후 상품개선 알림 모달
복사 시 자동으로 상품개선 DB에서 매칭 항목을 확인하여 모달 표시.

#### 알림 대상
1. **이슈관리 사진**: 발주 상품의 SKU에 이슈관리 등록 사진이 있으면 표시
2. **상품개선 항목**: 유형이 `재수배` 또는 `상품문제`이고 상태가 `시작전` 또는 `처리중`인 항목

#### 상품명 키워드 매칭 로직
- 상품개선 상품명을 공백 기준으로 키워드 분리
- 키워드의 **70% 이상**이 발주신청 상품명(노출상품명+옵션명)에 포함되면 매칭
- 예: 상품개선 `"하루모음 핸드크림 세트"` (3키워드) → 발주신청 상품명에 2개 이상 포함 시 매칭

#### 모달 표시
- 이슈관리 사진: 파란색 배지 + 사진 표시
- 상품개선 항목: 보라색 배지 + 유형명(재수배/상품문제) 표시 + 사진(있으면)
- 좌우 화살표로 항목별 이동 가능

| 총 수량 | 발주량 합계 |

## 데이터 흐름

```
재고계산기 시트 (CN발주량 > 0)
  + 쿠팡바코드 시트 (브랜드, 센터, 비고)
  + 발주서 양식 시트 (비고 우선)
  + 특별관리 시트 (봉제, FBC, 기타1회성)
  → 발주번호 자동 생성 + 테이블 표시

복사/다운로드 시:
  + 이슈관리 사진 DB (issue_img_data)
  + 상품개선 DB (improvement_items) — 재수배/상품문제 + 시작전/처리중
  → 키워드 70% 매칭 → 알림 모달 표시
```
