## 마감 상품 기능

### 개요
재고관리 > 마감 상품 메뉴에서 장기재고 키워드를 등록하고, 홈 대시보드에서 장기재고를 분리 계산하는 기능.

### 구조

**페이지**: `/inventory/closed` → `src/pages/ClosedProducts.jsx`

**DB 키**: `closed_products`
```json
[
  {
    "keyword": "키워드",
    "products": [
      { "barcode": "xxx", "name": "상품명", "stock": 100 }
    ]
  }
]
```

### 마감 상품 페이지 기능
1. 상품명 키워드 입력 + 추가 버튼
2. 키워드 등록 시 재고계산기 시트(TSV_CALC, gid=1349677364)에서 상품명 매칭
   - cols[2]: 바코드, cols[3]: 상품명, cols[14]: 총재고
3. 매칭된 상품들을 바코드/상품명/재고로 DB에 저장
4. 키워드별 상품 테이블 표시 + 삭제 기능
5. 새로고침 버튼으로 시트 데이터 재조회

### 홈 대시보드 계산 로직 (Home.jsx)

**변경 전**: 가용재고 / 악성재고 (최종마감·마감대상 기준 분리)
**변경 후**: 가용재고 / 장기재고 (키워드 기준 분리)

| 메트릭 | 계산식 |
|--------|--------|
| 가용재고 | 전체 재고 - 장기재고 |
| 가용재고 총 원가 | 전체 원가 - 장기재고 원가 |
| 장기재고 | closed_products 키워드가 상품명에 포함되는 상품의 총재고 합 |
| 장기재고 총 원가 | 장기재고 수량 × 바코드별 원가(쿠팡바코드 시트) |

**홈화면 표시 순서**: 월 품절률 / 가용 재고 / 가용 재고 총 원가 / 장기 재고 / 장기 재고 총 원가

### 데이터 흐름
1. 마감 상품 페이지에서 키워드 등록 → DB(closed_products) 저장
2. 홈화면 로드 시 DB에서 키워드 조회 → 재고계산기 시트 상품명과 매칭
3. 매칭된 상품은 장기재고, 나머지는 가용재고로 계산

### 관련 파일
- `src/pages/ClosedProducts.jsx` — 마감 상품 페이지
- `src/pages/Home.jsx` — 홈 대시보드 (355~410줄 계산 로직)
- `src/components/Layout.jsx` — 메뉴 등록
- `src/App.jsx` — 라우트 등록
