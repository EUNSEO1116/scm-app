## 기능: NEW 품절현황 재고 수정

### 배경
업로드 엑셀에서 쿠팡재고가 0으로 잘못 기록되어 실제 재고가 있는 상품이 품절로 표시되는 문제.
사용자가 직접 재고를 정정하여 품절 목록과 품절률에서 제외하는 기능.

### 변경 파일
- `src/pages/SoldOutAnalysis.jsx`

### 세부 구현

#### 1. 재고계산기 G열 읽기
- `cMap`에 `calcStock` (cols[6]) 추가
- 기존에 읽지 않던 재고계산기의 쿠팡재고 값을 함께 저장

#### 2. 불일치 자동 감지
- 엑셀 `coupangStock === 0` AND 재고계산기 `calcStock > 0` → `mismatch: true`
- 업데이트 버튼 클릭 시 자동 비교

#### 3. 수정 버튼 UI
- 불일치 항목의 쿠팡재고 셀 옆에 파란색 `수정 (N)` 버튼 표시
- N = 재고계산기 실제 재고 수량
- 툴팁으로 상세 안내

#### 4. 수정 저장 및 반영
- DB `soldout_stock_corrections`에 optionId 영구 저장
- 품절 목록에서 즉시 제거
- 품절률 즉시 재계산 + 스냅샷 갱신
- 다음 업데이트 시에도 수정된 항목은 품절 판정 건너뜀
