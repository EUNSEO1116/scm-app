# 총재고원가 대시보드

## 로우데이터
- **시트명:** 쿠팡바코드
- **필요 컬럼:** 바코드, 원가, 상태

## 대시보드 위치
- HOME 화면 기존 "바로가기" 영역을 대체
- CSS 간격/레이아웃은 기존과 동일하게 유지 (flex: 1, 동일 영역)

## 대시보드 구성 (5개 항목)

| 항목 | 단위 | 설명 | 색상 |
|------|------|------|------|
| 월 품절률 | % | 판매중 품절률 | 회색 (#f0f0f0) |
| 가용 재고 | 개 | 판매중 의 총 재고 | 파란 파스텔 (#dceefb) |
| 가용 재고 총 원가 | 원 | 판매중 재고의 총 원가 | 파란 파스텔 (#dceefb) |
| 악성 재고 | 개 | 최종마감 수량 | 빨간 파스텔 (#fde8e8) |
| 악성 재고 원가 | 원 | 최종마감 총 원가 합 | 빨간 파스텔 (#fde8e8) |

## 디자인
- 테이블 형태: 헤더(컬럼명) / 값 / 설명 3행 구조
- 가용 재고 영역: 파란 파스텔톤
- 악성 재고 영역: 빨간 파스텔톤
- 월 품절률: 중립 회색
