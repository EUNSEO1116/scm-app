# FBC관리 (Fulfillment By Coupang)

## 개요
쿠팡 풀필먼트 물류비용 비교, 절감 분석, FBC 품목 관리 기능군.

---

## 1. FBC 비용 계산기 (FbcCalculator.jsx)
**경로:** `/fbc`

### 기능
- FBC 출고내역서(엑셀) 업로드 → 비용 자동 계산
- 일반배송 vs FBC 밀크런 비용 비교
- 계산 결과를 localStorage에 저장 → 대시보드 연동

### 처리 단계
1. ① 서류 업로드 — 엑셀 파일 선택
2. ② 서류 분석 결과 — 묶음별 박스/팔레트 분석
3. ③ 비용 단가 설정 — 차량/밀크런 단가 조정
4. ④ 박스 크기별 팔레트 적재 설정
5. ⑤ 비용 비교 결과 — 절감액/절감률 표시

### 비용 계산식
```
[일반배송]
  차량비: CBM별 선택 (1톤 ~ 3.5톤장축)
  배송비: 박스수 × 5,000원
  합계 = 차량비 + 배송비

[FBC 밀크런]
  밀크런: 25,800원(1팔) / 51,600원(2팔) / 73,530원(3팔) / 91,460원(4팔+)
  작업비: 팔레트수 × 35,000원
  합계 = 밀크런비 + 작업비

[절감액] = 일반배송 합계 - FBC 합계
```

### 엑셀 파싱
- 각 시트 = 하나의 묶음
- 셀 배경색으로 팔레트 그룹 구분
- 박스 크기, 수량, SKU, CBM 자동 추출
- 파일명에서 날짜(YYMMDD) 파싱

### 데이터 저장
- localStorage `fbc_savings_history`에 계산 결과 누적 저장

### 연관 기능
- **FBC 대시보드** — 저장된 계산 결과를 시각화
- **홈** — FBC 절감 이벤트 표시

## 2. 절감 대시보드 (Dashboard.jsx)
**경로:** `/dashboard`

### 기능
- FBC vs 일반배송 절감 추이 분석
- 월별 차트 시각화
- 이력 검색/삭제/내보내기

### 통계 카드
| 항목 | 설명 |
|---|---|
| 총 누적 절감액 | 전체 기간 절감 합계 |
| 이번달 절감액 | 현재 월 절감 합계 |
| 건당 평균 절감 | 총 절감액 / 건수 |
| FBC 승률 | FBC가 저렴한 비율 (%) |

### 월별 차트
- 연도 선택 가능
- 양수/음수 바 차트
- 호버 시 상세 툴팁

### 이력 테이블
- 월별 그룹 (접기/펼치기)
- 검색: 파일명, 묶음명
- 컬럼: 날짜, 파일명, 묶음, 박스수, CBM, 일반비용, FBC비용, 절감액, 절감률
- 상세 모달: 비용 내역 표시
- 월별/전체 엑셀 내보내기
- 개별 삭제

### 데이터 소스
- localStorage `fbc_savings_history` (FBC 계산기에서 저장)

## 3. FBC 품목 (FbcItems.jsx)
**경로:** `/fbc/items`

### 기능
- Google Sheets '품목 FBC' 시트에서 FBC 품목 관리
- 출고내역서 업로드로 품목 정보 자동 갱신
- 일괄 편집 기능

### 상태 추적
| 상태 | 포함 단계 |
|---|---|
| 🛒 구매중 | 업체발송대기, 내륙운송중 |
| 🔧 작업중 | CN 창고도착, 작업대기, 출고대기 |
| 🚚 출고중 | 출고완료 |

### 주요 컬럼
- 쿠팡바코드, 상품명, 옵션명
- 1팔레트 물량, 박스당 수량, CBM
- 상자 크기1/2, 팔레트 박스제한

### 출고내역서 업로드
- 박스 크기, CBM, 수량 자동 추출
- 기존 FBC 품목과 병합
- 신규/변경 항목 하이라이트

### 일괄 편집
- 다중 선택 → 여러 필드 동시 수정 → 일괄 적용

### 엑셀 내보내기
- 헤더 서식, 폰트, 정렬 포함
- Google Sheets 업로드용
