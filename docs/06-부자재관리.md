# 부자재관리

## 개요
부자재(포장재, 소모품 등) 재고 관리 및 발주 추적 기능.

---

## 1. 부자재 목록 (SuppliesList.jsx)
**경로:** `/supplies`

### 데이터 소스
- Google Sheets '부자재목록 최신' 시트 (CSV)
- localStorage `supplies_orders` (부자재 발주 데이터 참조)

### 주요 컬럼
| 컬럼 | 설명 |
|---|---|
| 분류 | 카테고리 (배지 표시) |
| 용도1, 용도2 | 사용처 |
| 이용중인 제품 | 해당 부자재 사용 상품 목록 (접기/펼치기) |
| CN상품명 | 중국 상품명 |
| 사이즈 | 규격 |
| CN바코드 | 중국 바코드 |
| MOQ | 최소주문수량 |
| 세트 | 세트 구성 (예: "4p") |
| 가격 | 단가 (원) |
| 남은수량 | 계산된 잔여 수량 |

### 남은수량 계산 로직
1. 부자재 발주(`supplies_orders`)에서 입고완료 수량 합산
2. 발주장부 시트에서 해당 바코드 사용량 조회
3. 입고량 - (발주수량 × 세트수량) = 남은수량
4. 부자재 입고일 이후의 발주만 차감

### 토글 컬럼
- "이용제품 ON/OFF" — 이용중인 제품 컬럼 표시/숨김
- "CN상품명 ON/OFF" — CN상품명 컬럼 표시/숨김

### 검색 및 필터
- 상품명/CN상품명/바코드/용도 텍스트 검색
- 분류 드롭다운 필터
- 컬럼 헤더 클릭 정렬 (오름차순/내림차순)

### 연관 기능
- **부자재 발주** — `supplies_orders` 데이터로 입고량 계산
- **발주장부** (`01-재고관리.md`) — 부자재 소모량 산출에 발주 데이터 사용

## 2. 부자재 발주 (SuppliesOrder.jsx)
**경로:** `/supplies/order`

### 기능
- 부자재 입고 주문 등록 및 추적
- 자동 입고 완료 처리
- 부자재 목록과 연동하여 잔여수량 반영

### 주문 등록 폼
| 필드 | 설명 |
|---|---|
| 발주번호 | 예: PO-001 |
| CN바코드 | 자동완성 (부자재 목록 기반) |
| 발주수량 | 주문 수량 |
| 예상입고예정일 | 도착 예정일 |

### 주문 상태
| 상태 | 색상 | 조건 |
|---|---|---|
| 입고대기 | 주황 | 예정일 이전 |
| 입고완료 | 파랑 | 예정일 경과 (자동 전환) |
| 소진 | 회색 | 남은수량 ≤ 0 |

### 자동 입고 처리
- 예상입고예정일 경과 시 자동으로 "입고완료" 상태 전환
- 컴포넌트 업데이트 시 로직 실행

### 요약 통계
- 입고대기: X건 / 입고완료: Y건 / 전체: Z건

### 검색
- 발주번호, 바코드, 상품명으로 검색

### 데이터 저장
- localStorage `supplies_orders`

### 연관 기능
- **부자재 목록** — 바코드 매칭으로 잔여수량 계산에 사용됨
