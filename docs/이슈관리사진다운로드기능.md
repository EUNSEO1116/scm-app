# 이슈관리 사진 다운로드 기능

## 개요
이슈관리 페이지에서 등록된 모든 사진을 상품명 폴더별로 정리된 ZIP 파일로 다운로드하는 기능.

## 구현 내용

### 사용 라이브러리
- **JSZip**: ZIP 파일 생성
- **file-saver**: 브라우저에서 파일 다운로드 트리거

### 동작 방식
1. 필터 바의 `📥 사진 다운로드 (N)` 버튼 클릭
2. DB(`issue_img_data`)에서 전체 이미지 데이터 로드
3. 바코드 → 상품명 매핑 (스프레드시트 데이터 기반)
4. 상품명별 폴더 생성, base64 이미지를 JPG 파일로 변환
5. ZIP 파일 다운로드: `이슈관리_사진_YYYY-MM-DD.zip`

### ZIP 파일 구조
```
이슈관리_사진_2026-04-10.zip
├── 브랜드A 상품명1/
│   ├── 사진1.jpg
│   ├── 사진2.jpg
│   └── 사진3.jpg
├── 브랜드B 상품명2/
│   └── 사진1.jpg
└── ...
```

### 버튼 상태
- 사진이 없으면 비활성화 (disabled)
- 다운로드 중일 때 `⏳ 다운로드 중...` 표시
- 괄호 안 숫자: 사진이 등록된 상품 수
